# Mine-Dine — Project Management

## Single source of truth
- Roadmap + backlog: `docs/pm/BACKLOG.md`
- Current sprint: `docs/pm/SPRINT.md`
- Decisions: `docs/pm/DECISIONS.md`
- Operating rules: `docs/pm/RULES.md`

## Cadence
- Daily: move 1 card (Backlog → Sprint → Done)
- Weekly: pick 1 theme + 3 shippable tasks

## Definitions
- **Done** = deployed or merged + documented + tested (where relevant)
- **Small** = can be finished in ≤ 90 minutes
