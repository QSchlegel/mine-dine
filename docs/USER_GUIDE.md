# User Guide

## Getting Started

### As a Guest

1. **Sign Up/Sign In**: Create an account or sign in with email/password or OAuth (Google/GitHub)
2. **Create Profile**: Add your name, bio, and photos
3. **Select Tags**: Choose tags that describe your interests
4. **Discover Hosts**: Use the swipe interface to find hosts
5. **Book Dinners**: Browse and book available dinners
6. **Leave Reviews**: Share your experience after attending

### As a Host

1. **Apply to Host**: Submit an application to become a host
2. **Wait for Approval**: Admin will review your application
3. **Create Dinners**: Once approved, create dinner listings
4. **Set Pricing**: Configure base price and optional add-ons
5. **Upload Bills**: Add grocery bills for transparency
6. **Manage Bookings**: View and manage guest bookings

## Features

### Swipe Interface

- Swipe right to like a host
- Swipe left to pass
- Get matched when both parties like each other

### Booking Process

1. Select a dinner
2. Choose number of guests
3. Select optional add-ons
4. Complete payment via Stripe
5. Receive booking confirmation

### Messaging

- Message hosts directly
- Messages can be linked to bookings
- Real-time conversation interface

### Reviews

- Rate dinners 1-5 stars
- Add optional comments
- Reviews are visible on dinner pages

## Pricing

- Base price: 50€ per person (default, hosts can customize)
- Add-ons: Optional services with custom pricing
- Total = (Base × Guests) + Add-ons
